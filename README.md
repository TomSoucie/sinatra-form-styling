# Sinatra, ActiveRecord, RSpec Skeleton

Built off [this](https://github.com/turingschool-examples/sinatra-active-record-skeleton).

To setup, clone this down this repo, `cd` into it and:

```
bundle install
rake db:setup
shotgun
```
